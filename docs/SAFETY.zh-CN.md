# 安全指南（极限竞速：地平线 6）

本文说明 forza-painter FH6 对您电脑的作用，以及如何降低使用风险。

## 导入前

1. **保持在 Vinyl Group Editor（贴膜组编辑器）** — 在 FH6 中打开「创建贴膜组」/「贴膜组编辑器」，导入期间不要切换菜单。
2. **解除模板分组** — 已分组的贴膜会导致图层写入失败。
3. **确认图层数量** — 在应用中填写游戏内模板的确切图层数（FH6 必填）。
4. **使用匹配 JSON** — 低图层 JSON 导入高图层模板会显得模糊。

## 点击导入时会发生什么

1. 应用会显示 **同意** 摘要：目标进程，以及仅修改贴膜编辑器数据。
2. 若尚未获得 **管理员** 权限，应用会请求 **以管理员身份重启**（一次 UAC）。这是为了 `OpenProcess` 访问 FH6，离线生成图片不需要。
3. 应用可能会 **自动定位** FH6 图层表（只读扫描）并在写入前验证。
4. **辅助任务**（同一程序，helper 模式）执行导入；日志会显示启动了哪个辅助任务。

## 不会修改的内容

本工具不会修改速度、金钱、车辆、比赛成绩或在线数据，仅写入编辑器使用的贴膜图层字段。

## 账号 / 封禁风险

在 FH6 运行时读写游戏内存与部分外部内存工具同类。即使仅修改外观，微软仍可能按政策或反作弊处理。

**风险自负。** 作者不对账号处罚负责。

## 推荐流程

| 步骤 | 需要游戏运行？ | 需要管理员？ |
| --- | --- | --- |
| 生成 JSON（GPU） | 可选（关闭 FH6 可降低 GPU 发热） | 否 |
| 预览 / 滤镜 | 否 | 否 |
| 导入 FH6 | **是**（编辑器打开） | **是**（提示时） |
| 从 FH6 导出 | **是** | **是**（提示时） |

## 工具 → FH6 诊断

快照、对比、表检查会扫描游戏内存。**除非您了解内存调试，否则请勿使用。** 正常导入不需要这些功能。

## 若导入失败并提示「拒绝访问」

1. 接受提升权限提示，让应用以管理员身份重启。
2. 再次尝试导入。
3. 关闭其他可能占用 FH6 进程的内存工具。
4. 确认仍在 Vinyl Group Editor，且进程列表中的 PID 正确。

## experiments 文件夹

`experiments/` 下的脚本（例如完整图层 blob 恢复）**不支持**日常使用，可能导致 FH6 崩溃。请使用主程序导入流程。
